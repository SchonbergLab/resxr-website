# Installation

## Requirements
- Unity ( ResXR currently targets Unity 6)
- Meta Quest headset (Quest 2 / Quest Pro / Quest 3)
- Basic Unity + C# knowledge

## Open the project
1. Clone the Repository
1. Open the project in Unity
2. Let Unity resolve packages (Meta XR SDK will be pulled automatically)
3. Open a Base Scene (see "Base Scene Explained")
